# Проект: Место

Проект создан, чтобы делиться красивыми местами России.
Проект представляет собой одностраничный адаптивный сайт с применением форм и Javascript.

## В проекте используется:

- HTML
- CSS
- JS
- Figma
- Адаптивная верстка
- Grid Layout
- Forms
